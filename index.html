About the Application

The Srophé Application is an open source TEI publishing framework. Originally developed as a digital publication platform for Syriaca.org [http://syriaca.org/] the Srophé software has been generalized for use with any TEI publications.

The software features multilingual browse and search options, faceted search and browse, maps, RDF integration and more.

  • Timelines (https://timeline.knightlab.com/)
  • D3js visualizations
  • Content Negotiation module
    • HTML
    • TEI
    • geoJSON/KML
    • JSONLD
    • RDF/XML
    • RDF/TTL
    • Atom
    • PDF
    • Plain text
